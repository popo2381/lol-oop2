# League of Legends - 콘솔 전투 시뮬레이터
Java 콘솔 환경에서 동작하는 LoL 챔피언 전투 시뮬레이터입니다.


챔피언 선택/랜덤 소환, 랜덤 스킬 사용, 전투 로그 출력, 레벨업/부활 시스템을 포함합니다. 

**주요 기능**

#### + 게임 메뉴 및 모드 선택

+ 콘솔 메뉴 출력 후 모드 선택
  + **Normal Game** (1:1)
  + **Personal Game** (개인전/난투)
  + Team Game / 정보 조회 메뉴는 현재 보류(스켈레톤) 

#### + Normal Game (일반전 1:1)
 + 내 챔피언(MY) 1명과 상대 챔피언(ENEMY) 1명을 직접 선택하여 전투 진행 

 + 매 턴 선공은 랜덤(Random.nextBoolean()) 

 + 사망 시 부활 로직 수행 및 라운드 증가 

 + 한쪽이 더 이상 부활 불가가 되면 전투 종료 및 승자 출력 

#### + Personal Game (개인전/난투)
  + 입력한 인원수만큼(최대 8인) 챔피언을 랜덤 소환 

    + **1명은 MY**, 나머지는 ENEMY로 IFF 부여 

  + 매 턴 공격자/피격자를 랜덤 선택(동일 객체면 재선정) 

  + 전투 로그 출력 및 종료 시 우승자(마지막 남은 1인) 출력 

  + 전투 횟수(battleCount) / 생성된 챔피언 수(createdCount) 출력 

#### + 전투 시스템 (공통)
  + 챔피언은 공격 시 스킬 0~4를 랜덤 선택하여 사용 (random.nextInt(5))

  + 스킬은 basicAttack 또는 Q/W/E/R로 분기 

  + 피해량 = (공격자 ATK + 스킬 데미지) - 대상 DEF (최소 0 보정) 

#### + 레벨/경험치 시스템
  + 공격/피해 과정에서 경험치를 누적하고, 일정 조건에서 레벨업(HP/ATK/DEF 상승) 

#### + 부활 시스템
  + 챔피언은 최대 부활 횟수(5회)를 초과하면 isResurrectAvailable = false 처리 

  + NORMAL 모드에서는 부활 시 리스폰 딜레이(2초) 출력 

  + PERSONAL 모드에서는 사망 시점에 부활 로직을 타도록 처리 

#### + 로그/통계
  + 전투 시작 로그 출력(GameLog.LogEntry) 

  + 전체 로그 저장/출력/초기화 지원(GameLog.add/printAll/clear) 

  + 전투 횟수 카운트(BattleUtils.battleCount) 

### 프로젝트 구조
```
src/
 ├── Main.java            // 게임 메뉴/모드 실행, 입력 처리, 전투 루프
 ├── Champion.java         // 공통 스탯/전투/부활/레벨업 로직(추상 클래스)
 ├── Irelia.java           // Irelia 스킬 구현(Q/W/E/R)
 ├── Riven.java            // Riven 스킬 구현(Q/W/E/R)
 ├── Ezreal.java           // Ezreal 스킬 구현(Q/W/E/R)
 ├── Sona.java             // Sona 스킬 구현(Q/W/E/R), heal 예시 포함
 ├── Iff.java              // 진영 구분 ENUM (MY, ENEMY)
 ├── GameConstants.java    // 게임 상수(기본 스탯, 최대 레벨 등) 및 GAME_MODE
 ├── BattleUtils.java      // 전투 관련 유틸(전투 횟수, HP 비교)
 └── GameLog.java          // 전투 로그 저장/출력
```
### 실행 흐름
1.  프로그램 실행 시 게임 메뉴를 출력한다. 

2.  메뉴 번호 입력 후, 선택한 게임 모드에 따라 분기한다. 

3.  Normal: 내/상대 챔피언을 선택하고 1:1 전투를 진행한다. 

4.  Personal: 참가 인원수를 입력하고 랜덤 소환 후 난투 전투를 진행한다. 

5.  전투는 공격/피해/사망/부활을 반복하며, 조건 충족 시 종료 후 결과를 출력한다. 

 

### 개발 포인트
  + 추상 클래스 + 오버라이딩으로 챔피언별 스킬 구현 (useQ/W/E/R) 

  + Random 기반 전투 흐름(선공/스킬/대상 랜덤) 

  + 모드별 규칙 분기(GameConstants.GAME_MODE) 

  + Stream 활용: MY 챔피언만 필터링 출력
### 실행 예시 (콘솔 출력 샘플)
#### 1️⃣ 게임 시작 및 메뉴 선택
```
========================================
        League of Legends - GAME MENU
========================================
 1. 일반 게임 (Normal Game)
 2. 개인전 모드 (Personal Game)
 3. 팀전 모드 (Team Game)
----------------------------------------
 4. 챔피언 정보 조회
 5. 게임 정보 조회
----------------------------------------
 0. 게임 종료
========================================
메뉴 번호를 선택하세요: 2
```
#### 2️⃣ Personal Game – 챔피언 소환
```
========================================
    PERSONAL MODE는 개인전 모드 입니다.
========================================
         랜덤으로 진행되는 챔피언 목록
========================================
  1.Irelia  2.Riven  3.Ezreal  4.Sona
========================================
챔피언 인원수를 입력해주세요 (최대 8인): 8

나의 챔피언: MY
현재 참가중인 챔피언:
[[MY] irelia, [ENEMY] riven, [ENEMY] sona, [ENEMY] irelia2,
 [ENEMY] ezreal, [ENEMY] riven2, [ENEMY] sona2, [ENEMY] riven3]
```
#### 3️⃣ 전투 진행 예시
```
===== 1라운드 =====

[MY] irelia 이(가) Q 스킬을 사용했습니다.
[ENEMY] riven 의 HP가 72로 감소했습니다.

===== 2라운드 =====

[ENEMY] sona 이(가) R 스킬을 사용했습니다.
[MY] irelia 의 HP가 48로 감소했습니다.
```
#### 4️⃣ 사망 및 부활 처리
```
[ENEMY] ezreal 이(가) basic attack을 사용했습니다.
[ENEMY] sona 의 HP가 0이 되었습니다.
[ENEMY] sona 이(가) 부활합니다. (남은 부활 횟수: 4)
```
#### 5️⃣ 최후의 승자 출력
```
===== 17라운드 =====

[MY] irelia 이(가) R 스킬을 사용했습니다.
[ENEMY] riven3 의 HP가 0이 되었습니다.

[결과] 최후의 승자: [MY] irelia
```
